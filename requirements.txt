# RaspberryFluke Dependencies
# ============================================================
#
# Most dependencies are installed via apt on Raspberry Pi OS.
# Using apt-provided packages is strongly preferred over pip
# to avoid conflicts with the system Python environment.
#
# Run install.sh to handle all of these automatically.
#
# ============================================================
# Step 1 — System packages (handled by install.sh)
# ============================================================
#
#   sudo apt install -y \
#       git \
#       python3 \
#       python3-pip \
#       python3-pil \
#       python3-lgpio \
#       python3-rpi.gpio \
#       fonts-dejavu-core
#
# Note: lldpd is NOT required. RaspberryFluke uses raw AF_PACKET
# sockets to capture LLDP and CDP frames directly from the wire.
#
# ============================================================
# Step 2 — Waveshare e-paper library (handled by install.sh)
# ============================================================
#
# The Waveshare e-paper library is not available on PyPI.
# install.sh clones it from GitHub and copies the driver into
# the project directory automatically.
#
# Source: https://github.com/waveshare/e-Paper
# Required path inside the repo:
#   RaspberryPi_JetsonNano/python/lib/waveshare_epd/
#
# ============================================================
# Step 3 — Waveshare LCD library (optional, LCD display only)
# ============================================================
#
# Only required if DISPLAY_TYPE = "lcd" in rfconfig.py.
# The e-paper display is the recommended and default display.
#
# ============================================================
# Python version
# ============================================================
#
# Requires Python 3.9 or later.
# Raspberry Pi OS Bullseye and Bookworm both ship Python 3.9+.
#
# ============================================================
# pip-installable equivalents (reference only)
# ============================================================
#
# These are provided as a version reference.
# On Raspberry Pi OS, always use the apt versions above.

Pillow>=9.0.0
RPi.GPIO>=0.7.0
